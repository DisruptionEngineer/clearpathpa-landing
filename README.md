# Airgap Landing

Marketing landing page for Airgap Gate — AI-powered prior authorization intelligence for healthcare providers.

## Tech Stack

- [Astro](https://astro.build) v5 — static site generator
- [Tailwind CSS](https://tailwindcss.com) v4 — utility-first styling

## Development

```bash
npm install
npm run dev      # Start dev server at localhost:4321
npm run build    # Build for production
npm run preview  # Preview production build
```

## Deployment

Deployed to [Vercel](https://vercel.com) as a static site.
